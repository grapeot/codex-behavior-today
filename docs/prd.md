# PRD

## 产品目标

公开展示 Codex endpoint 在固定短 prompt 上的日常行为分布是否偏离自己的历史范围。

## 用户问题

今天的 Codex 行为变了吗？如果变了，是哪个固定 probe 的分布改变了，以及这种变化相对过去的自然波动有多异常？

## 非目标

不鉴定模型身份，不推断蒸馏或量化，不评估一般能力，也不将任何异常归因于供应商意图。

## 成功标准

- 每日运行采集 100 个成功样本或明确发布 partial 状态。
- 页面展示最新状态、日环比、历史基线 drift 和 cell 级别变化。
- 公开 Git 不包含 raw response、OAuth metadata 或本机运行信息。
- 每个公开指标可由 `data/daily/` 的聚合计数重算。
